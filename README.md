# 基于共现提取《釜山行》人物关系


《釜山行》是一部丧尸灾难片，其人物少、关系简单，非常适合我们学习文本处理。这个项目将介绍共现在关系中的提取，使用 python3 编写代码实现对《釜山行》文本的人物关系提取，最终利用Gephi软件对提取的人物关系绘制人物关系图。


* 釜山行中文文本 busan.txt
* 人物名词           dict.txt
* 主要程序           busan.py
* 提取结果（节点）   busan_node.txt 
* 提取关系结果（边） busan_edge.txt
